# Cadastro de dados (HTML + CSS + JS)

Abra o arquivo `index.html` no navegador e preencha a tabela de:

- **Duty Cycle**: 5, 10, ..., 100
- **LEDs**: verde, vermelho, azul
- **Campos**: comprimento de onda de pico (nm) e intensidade
- **Equipamentos**: OSA Visível e ThorLabs

## Observações

- Os dados são **salvos automaticamente** no navegador (localStorage).
- Use **Exportar CSV** para baixar os dados do equipamento selecionado.
- Use **Exportar JSON** para baixar todos os dados.
- Use **Importar JSON** para recuperar dados exportados.
